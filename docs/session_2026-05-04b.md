### Halation (`grade.fx`)
- **Gate threshold too high**: lower bound was `0.62`, dark-scene eff_p75 Ôëł 0.25 Ôćĺ gate never opened. Fixed: lower bound Ôćĺ `0.35`.
- **Ring artifact**: gate smoothstep created a hard radius boundary around lamps. Fix: removed gate entirely from the final multiply. `hal_delta` (blurred Ôłĺ sharp) is self-limiting ÔÇö zero in flat areas, nonzero only near bright-dark transitions. No gate needed.

### Pro-Mist (`pro_mist.fx`)

## Inverse grade pivot fix (`inverse_grade.fx` + `analysis_frame.fx`)

**Problem**: `lab.yz *= slope_factor` uses C=0 as pivot Ôćĺ all coloured pixels gain chroma uniformly (net saturation increase, not range expansion).

**Fix**: pivot expansion around scene mean chroma.
- `analysis_frame.fx`: added `RGBtoOklab` helper, `MeanChromaTex` (1├Ś1 RGBA16F), and `MeanChromaPS` pass (pass 9). Averages Oklab C over 32├Ś18 downsample for pixels with C > 0.05, EMA-smoothed (~12 frames at 60fps).
- `inverse_grade.fx`: declares same `MeanChromaTex`. Reads `mean_C`. Replaces `lab.yz *= factor` with:
  ```hlsl
  float new_C = mean_C + (C - mean_C) * factor;
  float2 dir  = lab.yz / max(C, 1e-5);
  lab.yz      = dir * max(new_C, 0.0);
  ```
  High-C pixels push further above mean; low-C pixels push further below. Mean chroma preserved.
- `grade.fx`: `chroma_str_base` reverted from coupled `0.085 - 0.075 * INVERSE_STRENGTH` to flat `0.075` (slight reduction from calibrated 0.085 to account for stronger inverse_grade at 0.60).
