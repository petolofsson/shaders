### R126 — FilmCurveApply body S-curve

**Result: PARTIAL — formula mismatch**

Return structure matches job spec:  
grade.fx:158: `return x + body_s - factor * shoulder_w * above * above + toe_fac * toe_w * below * below;` ✓

`body_s` formula **does NOT match job spec**:  
- Job spec says: `body_s = x * (1.0 - x) * (1.0 - 2.0 * x) * 0.12`  
- Actual code (grade.fx:156–157):
  ```hlsl
  float3 xw     = x * (1.0 - x);
  float3 body_s = max(0.0, xw * xw * (2.0 * x - 1.0)) * 0.65;
  ```
  Expands to: `max(0, (x(1-x))² × (2x-1)) × 0.65`

These are structurally different: the job spec formula is linear in `x(1-x)` and negative
for x>0.5 (lower-mid lift); the actual code is quadratic in `x(1-x)`, one-sided positive
for x>0.5 (upper-mid lift, matching CLAUDE.md description). The actual code matches
CLAUDE.md: "upper-mid one-sided body lift `max(0,(x(1-x))²(2x-1))*0.65`". The job spec
R126 formula is stale/incorrect. **Job spec needs updating.**
