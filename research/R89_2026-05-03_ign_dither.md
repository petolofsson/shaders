# Research Findings — IGN Blue-Noise Dither — 2026-05-03

## Status: Implemented — `grade.fx` (ColorTransformPS)

---

## Problem

The previous dither in `ColorTransformPS` used:

```hlsl
float dither = sin(dot(pos.xy, float2(12.9898, 78.233))) * 43758.5453 - floor(...);
```

This is white noise — energy is distributed uniformly across all spatial frequencies.
White noise dither leaves visible banding in smooth gradients (fog, sky, dark shadows)
because the brain integrates the quantization error across spatial frequency bands
equally, and the low-frequency component is perceptible.

---

## Physical / mathematical basis

Jimenez (2016) Interleaved Gradient Noise (IGN) is a blue-noise dither pattern —
it concentrates quantization error at high spatial frequencies (fine grain), pushing
error away from the low-frequency band where the human visual system is most
sensitive.

The IGN formula generates a stationary, spatially decorrelated pattern with a
spectrally blue power distribution:

```hlsl
float ign(float2 pos) {
    return frac(52.9829189 * frac(dot(pos, float2(0.06711056, 0.00583715))));
}
```

For animated content, the pattern can be temporally rotated by a frame-index offset
(golden-ratio sequence) to further spread quantization energy across time.

Blue noise has the property that adjacent pixels carry uncorrelated error — no
spatial structure is introduced — while the integrated RMS error over any large
region matches the theoretical quantization step. This makes it spectrally
equivalent to random noise at the scene level while appearing "grainier" (less
structured) than white noise at the pixel level.

---

## Implementation

`grade.fx` — `ColorTransformPS`, dither block:

```hlsl
// Before:
float noise = frac(sin(dot(pos.xy, float2(12.9898, 78.233))) * 43758.5453);
float dither = noise - 0.5;

// After (IGN):
float dither = frac(52.9829189 * frac(dot(pos.xy, float2(0.06711056, 0.00583715)))) - 0.5;
```

Applied as `lin += dither * (1.0 / 255.0)` before the 8-bit BackBuffer write.

Note: `ProMistPS` also uses this IGN formula (inherited during the pro_mist.fx →
grade.fx merge, 2026-05-05). R92 (IGN for ProMistPS) was therefore resolved by
the merge with no additional code change.

---

## Effect

Banding in smooth gradients — fog, sky gradients, dark shadow ramps — is replaced
by fine-grain noise with no visible structure. The dither amplitude (1/255) is
identical; only the spectral distribution changes.

---

## GPU cost

Same ALU as sin-based dither (both are ~4 instructions). Negligible difference.
