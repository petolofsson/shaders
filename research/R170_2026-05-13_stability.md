**Advisory 1 — `atan2(0,0)` in inverse_grade.fx:46:**
`float hue = frac(atan2(lab.z, lab.y) / 6.28318530)` — no zero-vector guard. Called when
C≈0 (achromatic). The result is gated by `c_weight = saturate((C - c_gate) / 0.15)` downstream,
so wrong values don't reach the output. On GPU hardware `atan2(0,0)` returns 0 (defined by GLSL/HLSL
specs as implementation-defined but universally 0 in practice). Low risk; no fix required.

**Advisory 2 — `atan2(0,0)` in analysis_frame.fx:195:**
`atan2(ab.y, ab.x)` for HWY_CHROMA_ANGLE highway write when scene is fully achromatic.
Same hardware behaviour applies. Highway consumers use the angle directionally (dot products),
so 0 is a safe fallback. Low risk; no fix required.

No `tex2Dlod(BackBuffer, ...)` found anywhere — confirmed. All BackBuffer reads use `tex2D`. ✓
