**R126 FilmCurve body S-curve:**

| Check | Result |
|-------|--------|
| `body_s = x*(1-x)*(1-2x)*0.12` | ✗ **DOES NOT MATCH** |
| Return: `x + body_s - factor*shoulder_w*above*above + toe_fac*toe_w*below*below` | ✗ **DOES NOT MATCH** |

Current formula (grade.fx:184–191):
```hlsl
float3 xs     = saturate(x);
float3 xw     = xs * (1.0 - xs);
float3 body_s = max(0.0, xw * xw * (2.0 * xs - 1.0)) * 0.65;
float3 sh_comp = above * above / (headroom + above);          // rational shoulder
float3 tc_comp = (0.06 / ktoe_rgb) * below * below / (ktoe_rgb + below);  // rational toe
return x + body_s - sh_comp + tc_comp;
```

The R126 spec formula `x*(1-x)*(1-2x)*0.12` produces a linear-quadratic lift. The current `xw²*(2xs-1)*0.65` is quadratic-quartic, zero for x≤0.5, peaks at x≈0.72 — the same upper-midrange intent but with stronger suppression of the lower half and a higher coefficient. The rational shoulder/toe (`sh_comp`, `tc_comp`) replace the `factor * shoulder_w * above²` spec form. This appears to be intentional post-R126 refinement (consistent with grade.fx comment lines 182–183). **Superseded, not a bug.**
