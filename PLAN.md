## Score reasoning

**Stage 0 — Input (92% novel)**
No game post-processing pipeline attempts to undo tonemapper compression. The per-hue HueCeil ceilings, R156 hue-specific slope bias, R163 dominant-hue alignment, R165 illuminant warmth CCT proxy, R193 toe_inv expansion form (matching ACES toe_inv exactly) — none of this exists in any game engine. R187 zero-anchored expansion (`C × factor`) with `(1 − lab.x)` continuous luma weight correctly reflects that highlights cannot recover chroma due to gamut geometry (cinema mastering data: highlight ΔC = −0.008, shadows/mids gain ~67%). Single-pass. The data highway architecture (HighwayTex 256×1 R16F, scene statistics across effects) is not a pattern in game post-processing. R198: FilmCurve pre-inverse (`FilmCurveInvCh`, piecewise exact) applied in `inverse_grade.fx` before chroma expansion — expansion now operates in the post-curve tonal domain. Remaining 1%: the chroma expansion slope `lerp(1.8, 1.15, saturate(median_C / 0.15))` treats all game tonemappers as equivalent — the slope range is calibrated against ACES compression specifically and would need re-derivation for Hejl, GT Tone Mapper, or custom curves. That is an honest ceiling on Stage 0 completeness. Drag: "chroma boost" as a concept exists in saturation knobs; SDR→HDR TV upconversion (Sony, Dolby) is conceptually adjacent, though not a game post-processing pattern.

**Stage 1 — Film Stock (94% novel)**
Games approximate film stocks with LUTs — even RDR2's celebrated film look is LUT-based. The 2383 3×3 spectral dye matrix from H-1-2383t primary data, H&D curve, DIR couplers, masking coupler, and chromatic floor are not found in any game post-processing pipeline. Rational film curve with histogram-mode-derived knee, adaptive print stock from p25/p75 scene percentiles — not in games. R191 P2: 3-way CC fires before print emulation (ACES LMT convention). R192 P3: PRINT_STOCK/BLEACH_BYPASS in `ApplyLook` post-chroma — correct physical LMT placement, pipeline order fully locked. Remaining 1%: calibration of the LMT position (PRINT_STOCK will feel stronger post-chroma; BLEACH_BYPASS shadow desaturation denser — values pending visual evaluation per HANDOFF). Drag: 3-way CC exists in UE5; S-curve tone shaping is universal; DIR-adjacent couplers exist in desktop photo tools (DxO FilmPack, Silver Efex) though not in real-time game pipelines.

**Stage 2 — Tonal (92% novel)**
Games have auto-exposure. Nothing else in this stage exists in game pipelines: zone system contrast from histogram statistics, Retinex illumination decomposition (2-scale: 1/16-res + 1/32-res illuminant), Oklab-stable L-substitution, intra-zone pixel variance driver, Munsell per-hue highlight rolloff from Renotation data, ambient shadow tint from scene illuminant. R190: guided filter replaces bilateral for LOCAL_TONE/CLARITY base layer — self-guided local linear model with adaptive ε (Hu et al. IET Image Processing 2023) at 1/8-res; eliminates bilateral lamp/window halos; content-adaptive (flat areas fully smooth, edges preserved ≤95.2%); fixed R190 log10→log2 mismatch that made LOCAL_TONE/CLARITY 3.32× wrong. R191 P1: Retinex before zone S-curve — equalized signal feeds contrast shaping, correct order. Remaining 2%: GF_EPS=0.05 is a fixed regularization parameter — content-adaptive ε scheduling against actual scene variance would be the principled completion; Retinex ceiling has no literature-derived upper bound. Drag: auto-exposure is structurally related to zone key; adaptive contrast exists conceptually in UE5 eye adaptation.

**Stage 3 — Color + Halation (92% novel)**
Purkinje shift, Abney effect, Hunt effect, Helmholtz-Kohlrausch, HELMLAB Fourier hue correction — none implemented in any commercial game engine. Halation as emulsion physics (DoG PSF, luma-neutral orange tint, sky suppression via exp(−(B−R)×7)) is not found in games; bloom is. Munsell Renotation gamut ceilings (HueCeil — per-hue maximum chroma from Renotation data) are not in games. R196-E: cusp-relative chroma compression — `d_norm = final_C / HueCeil(hue)`, Reinhard on excess above 0.85 maps [0.85,∞)→[0.85,1.0) asymptotically; replaces the hard HueCeil ceiling for problem hues (deep reds, cyans, yellow-greens) with a smooth compression matching the ACES2 principle in Oklab space; in-gamut pixels untouched (min()). Three effects scene-adaptive: (1) illuminant-adaptive halation rem-jet — G weights modulated by CAT16 warmth proxy; (2) scene-adaptive HK magnitude — `lerp(0.32, 0.18, zone_log_key / 0.50)` per Hellwig 2022; (3) scene-adaptive Abney scale — `1 + median_C × 0.25` per Pridmore 2007. Constants sourced from literature: HK `f_hk` and C^0.587 from Hellwig 2022 Fourier fit; Abney per-hue from Pridmore 2007 bimodal data; halation R:G:B ≈ 30:3:1 from emulsion spectral physics. Remaining 1%: g_mod scale 0.25 and Abney scene-scale 0.25 are conservatively calibrated — directionally supported but not directly measured against psychophysical data. Drag: hue rotation exists in UE5 and most grading stacks; basic saturation/vibrance is universal.

**Output — Diffusion + Grain (95% novel)**
Game grain is universally simple: white noise or single-octave blue noise with flat amplitude. The Selwyn 2383 granularity envelope `sqrt(1−L_gamma)` (perceived peak in upper shadows), per-channel dye layer sizing (G finest, B coarsest matched to 2383 cyan/magenta/yellow dye depth), single 24fps slot snap — none of this exists in any game. R196-F: half-res grain clustering — coarse GrainValueNoise snapped to 2px grid (`floor(pos*0.5)*2`), adjacent 2×2 blocks share the same sample, cubic interpolation between corners produces organic blobs instead of per-pixel speckling (eliminates digital sensor character). Fine blue-noise retained at full-res for sub-cluster texture. R168 physical halation DoG PSF with separate AH-layer rem-jet attenuation is an emulsion physics model, not bloom; no game implements it. HBM dual-component diffusion (additive shimmer + midtone overlay, separately gated) is not a pattern in game bloom. Polydisperse chromatic scatter per channel not in games. Remaining 2%: Selwyn amplitude calibration against actual 2383 densitometric measurements (current 0.018 coefficient is empirical); HBM radial PSF shape (xs=1.6, ys=0.08 oval) is observation-calibrated, not derived from measured diffusion PSF data. Drag: film grain and bloom both exist as concepts; the Gaussian blur chassis is universal.
